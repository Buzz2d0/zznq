如何使用API主动触发？

在 docs.github.com 中找到了手动触发的接口。

  1. 在Github Settings 处（即 https://github.com/settings/tokens ）申请Tokens，放置API header 中 Authorization: token {{your token}}

  2. 在仓库根目录下创建 .github/workflows/heartbeat.yml 文件：

name: HeartBeat

on:
  workflow_dispatch:
    inputs:
      value:
        description: 'value'
        required: false

jobs:
  sync:
    name: Build
    runs-on: ubuntu-latest
    steps:
      - name: Test-on-01
        run: |
                    echo ${{ github.event.inputs.value }}
  1. 请求接口（以我的仓库为例）：
curl \
  -X POST \
  -H "Accept: application/vnd.github.v3+json" \
  -H "Authorization: token xxxxxxxxx" \
  https://api.github.com/repos/Buzz2d0/Buzz2d0/actions/workflows/heartbeat.yml/dispatches \
  -d '{"ref":"main","inputs":{"value":"66"}}'
